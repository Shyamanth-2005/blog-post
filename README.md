# Blog Post Website

A responsive blog post website featuring modern CSS layout techniques and clean design.

## Project Overview

This project contains a complete blog post webpage with responsive design elements, showcasing various CSS techniques including Flexbox and CSS Grid layouts.

## Files Structure

```
├── index.html          # Main landing page
├── blog.html           # Blog post page
├── css-grid.html       # CSS Grid showcase
├── flexbox.html        # Flexbox showcase
├── style.css           # Main stylesheet
├── content.txt         # Sample content
└── img/                # Image assets
    ├── challenges.jpg
    ├── laura-jones.jpg
    ├── post-img.jpg
    ├── related-1.jpg
    ├── related-2.jpg
    └── related-3.jpg
```

## Features

- Responsive design that works on mobile, tablet, and desktop
- Modern CSS layout techniques (Flexbox and CSS Grid)
- Clean typography and spacing
- Attractive blog post layout with featured images
- Related posts section
- Author information display

## Pages

1. **Home Page (index.html)** - Landing page with featured content
2. **Blog Post (blog.html)** - Detailed blog post with content and sidebar
3. **CSS Grid Demo (css-grid.html)** - Showcase of CSS Grid capabilities
4. **Flexbox Demo (flexbox.html)** - Showcase of Flexbox capabilities

## Technologies Used

- HTML5
- CSS3 (with Flexbox and CSS Grid)
- Responsive design principles

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in your preferred browser
3. Navigate between pages using the navigation links

## Customization

To customize this template:
1. Modify `style.css` to change colors, fonts, and layout
2. Update content in HTML files as needed
3. Replace images in the `img/` folder with your own

## Browser Compatibility

This project uses modern CSS features and should work in all modern browsers (Chrome, Firefox, Safari, Edge).
